# MiniDevOpsHub Dashboard (Frontend)

This folder is for the dashboard UI. Recommended: React, Vite, or similar SPA framework.

## UI Sections
- Apps (control panel)
- Linked repository
- Deployments (version history, blue/green)
- Logs (live, scrollable)
- Workers (status, IP)
- Help (usage guide)

## API
See ../api/openapi.yaml for backend contract.

## Getting Started
- Use `npm create vite@latest` or `npx create-react-app` to scaffold.
- Set API base URL to `/api/` (relative path).
